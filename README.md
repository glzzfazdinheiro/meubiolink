# 🔗 Bio Link | glzz

Um hub de links pessoal com player de música integrado, animação de partículas e funcionalidades modernas — perfeito para usar como sua "árvore de links" no Instagram, TikTok ou GitHub. (Você pode adicionar mais quantos links e redes sociais que quiser)

## ✨ Funcionalidades

- ✅ **Tela de entrada** com efeito fade
- 🎨 **Design glassmorphism** com blur e bordas suaves
- 🎵 **Player de música minimalista** com:
  - Play/Pause
  - Mutar/Desmutar
  - Indicador da faixa atual
  - Toca automaticamente a próxima música da playlist
- 📋 **Botão do Discord** que:
  - Copia o username `inimigodapoliciacivil`
  - Mostra um toast flutuante próximo ao ícone
  - Abre um modal perguntando se quer abrir o Discord
  - Fallback para navegadores sem suporte à Clipboard API
- 🔗 Links para Instagram, TikTok e GitHub
- 🖼️ Avatar com fallback (caso a imagem não exista)
- ✨ **Partículas animadas** no fundo (canvas)
- 📱 Totalmente responsivo
- ♿ Acessibilidade com `aria-label`
- 🖱️ Efeito 3D Tilt no card (desktop)
- 🎯 Otimizações de performance (`lazy loading`, `preload`)

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Glassmorphism, media queries)
- JavaScript (Vanilla)
- [Font Awesome 6](https://fontawesome.com/) (ícones)
- [Google Fonts - Inter](https://fonts.google.com/specimen/Inter)
